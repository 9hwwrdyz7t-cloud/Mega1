# App Store Submission Checklist

Use this checklist before submitting Flezen Link Player to App Store Connect.

## Xcode project

- [ ] Bundle Identifier is unique and owned by you.
- [ ] Apple Developer Team is selected.
- [ ] Version and build number are set correctly.
- [ ] App icon is complete for all required sizes.
- [ ] Launch screen looks clean on iPhone and iPad.
- [ ] App builds in Release mode.
- [ ] App runs on a real iPhone.
- [ ] App runs on iPad or iPad simulator.

## Playback testing

- [ ] Direct MP4 link plays.
- [ ] Direct MP3 link plays.
- [ ] HLS `.m3u8` link plays.
- [ ] Invalid URL shows a friendly error.
- [ ] Favorites can be added and removed.
- [ ] Recent history saves correctly.
- [ ] Full-screen playback works.
- [ ] Background audio behavior is tested on a real device.
- [ ] AirPlay route appears when available.

## Legal and content policy

- [ ] App only plays user-provided links.
- [ ] App does not bypass DRM, paywalls, login pages, or access controls.
- [ ] App description clearly says users must only play media they are authorized to access.
- [ ] Any sample media streams are legal, public, and properly licensed.

## Privacy

- [ ] No analytics added unless disclosed.
- [ ] No ads added unless disclosed.
- [ ] No account system added unless disclosed.
- [ ] App Privacy answers match the app behavior.
- [ ] Privacy manifest remains accurate.

## App Store Connect

- [ ] App name: `Flezen Link Player`.
- [ ] Subtitle explains the app simply, e.g. `Play your authorized media links`.
- [ ] Category selected appropriately, likely Entertainment or Utilities depending on positioning.
- [ ] Screenshots uploaded for required devices.
- [ ] Description avoids claims about playing restricted or unauthorized content.
- [ ] Support URL is available.
- [ ] Privacy Policy URL is available if required by your final app behavior.

## Suggested App Store description

Flezen Link Player lets you paste and play compatible media links using the native iOS player experience. Save favorite links, reopen recent streams, and enjoy supported playback features such as full-screen viewing, AirPlay, and background audio where available.

Only play media that you own, have permission to access, or are otherwise authorized to stream.
